# golearn
go 语音自学
第一节 环境配置  
  使用mac去官网下载安装包即可，安装完后,打开命令工具  输入 >>>  go version 有反应则安装成功
  基本命令 
    go build 编译代码
    go run test.go 运行
  test.go 文件基本格式为
    package main     入口包，固定格式，程序入口
    import "fmt"     导包（基本包）
    func main(){     入口方法
      fmt.Println("hello world")   输出语句
    }
  ---------------------------------
  运行 go build------>  go run test.go  ----》hello world
